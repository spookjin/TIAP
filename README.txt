# Gerenciamento de Estoque — Base HTML/CSS/JS

Este é um ponto de partida **sem frameworks** (apenas HTML, CSS e JavaScript) inspirado no seu projeto com Next.js + shadcn.
Ele inclui:

- Listagem de produtos com busca, filtros, paginação, criar/editar/excluir (modal).
- Persistência local com `localStorage` (sem backend).
- Gestão simples de categorias.
- Indicadores básicos (relatórios).
- Exportação CSV.
- Tema claro/escuro.

## Como usar

1. Faça download do ZIP e extraia.
2. Abra `index.html` no seu navegador.

> Se quiser hospedar, basta subir os arquivos estáticos (ex.: GitHub Pages).

## Onde estender

- **API/Backend**: troque o `localStorage` por chamadas `fetch` para sua API.
- **Autenticação**: adicione tela de login/controle de acesso.
- **Movimentações de estoque**: crie uma página para entradas/saídas.
- **Relatórios**: conecte bibliotecas de gráfico (Chart.js) e endpoints reais.
- **Validações**: adicione regras específicas do seu domínio.

Boa sorte no trabalho! ✨
